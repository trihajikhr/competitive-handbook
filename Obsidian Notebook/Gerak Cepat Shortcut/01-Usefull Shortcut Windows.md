# Usefull Shortcut Windows

## 1 | Copy Paste

>  Ctrl + C, Ctrl + V, Windows + V

Untuk melakukan copy paste, ada cara klasik berupa Klik kanan, pilih `Copy`, lalu di file tempat kita ingin paste, klik kanan mouse lagi lalu `Paste`.

Ada juga yang lebih cepat, yaitu dengan menggunakan bantuan keyboard, yaitu dengan mengandalkan shortcut. Tekan saja `Ctrl + C`, dan di file kita ingin paste, tekan `Ctrl + V`.

> Kita berbicara Copy Paste teks, bukan file!

Nah, ada trick lagi dimana kita bisa menyimpan (*store*) beberapa clip teks yang kita copy, lalu ketika kita ingin paste, kita bisa menekan `Windows + V`, dan kita bisa memilih clip mana yang ingin kita paste. Ini mempercepat kita, jika ingin bolak-balik copy paste beberapa teks, namun hanya dengan memilih clip saja. Namun pastikan kita memasukan clip teks tersebut kedaftar clip, dengan cara `Copy` atau `Ctrl + C` terlebih dahulu.

## 2 | Navigasi & Manajemen Jendela

> Alt + Tab

Saat membuka beberapa aplikasi sekaligus, kadang kita berpindah ke jendela lain dengan cara mengklik icon aplikasi di Windows task bar dibagian bawah.

Namun, cara yang lebih cepat adalah dengan menggunakan shortcut `Alt + Tab`. Ketika shortcut ini ditekan, maka akan muncul daftar semua jendela aplikasi aktif. Untuk berpindah ke jendela lain, tahan `Alt`, dan tekan `Tab` berulang kali untuk memilih jendela yang ingin dibuka. Jika ingin berpindah antara 2 jendela saja berkali-kali, langsung tekan keduanya secara cepat. Aksi ini akan membuka jendela terakhir yang aktif.

## 3 | Navigasi Task View

> Windows + Tab

Windows versi modern menawarkan beberapa *desktop virtual* untuk memisahkan konteks kerja. Tombol ini ada dibagian bawah, tepatnya dibagian taskbar.

Untuk membukanya dan bernavigasi dengan cepat, tekan saja `Windows + Tab`. Selain membuka *desktop virtual*, aksi ini juga membuka daftar aplikasi yang sedang dibuka dan dijalankan di *desktop virtual* tersebut. Untuk barnavigasi antara *desktop virtual*, setelah `Windows + Tab`, lepaskan `Windows` dan tekan `Tab` beberapa kali. 

Kita juga bisa berpindah jendela aplikasi atau *desktop virtual* disini selain dengan `Tab`,  yaitu menggunakan `arrow pad`.

## 4 | Split Screen

> Windows + →/←/↓/↑

Untuk memsplit scree, kita biasanya mendrag bar dari jendela aplikasi keatas tengah windows, yang akan menawarkan beberapa opsi split screen. 

Tapi kita juga bisa menggunakan shortcut untuk mengatasi ini:
- `Windows + ↑/↓`: Maksimalkan atau minimalkan jendela;
- `Windows + →/←`: Pindahkan jendela ke sisi kiri/kanan layar.

## 5 | Hapus per Kata

> Ctrl + Backspace

Jika ingin menghapus seluruh kata alih-alih perkata, maka gunakan saja shortcut `Ctrl + Backspace`. Sangat berguna saat melakukan koreksi cepat tanpa harus menahan tombol `Backspace` lama-lama. 

Aku juga menggunakan trik ini untuk 10fastfinger, website latihan mengetik. Trik ini sangat berguna karena mempercepat koreksi kata yang salah.

## 6 | Select Teks

Kita biasanya melakukan select teks dengan cepat dengan cara drag mouse. Tapi selain menggunakan cara tersebut, ada cara yang lebih cepat dengan menggunakan shortcut apabila yang ingin di select hanyalah beberapa.

- `Shift + →/←`: Select huruf demi huruf
- `Shift + ↑/↓`: Select satu baris penuh keatas/kebawah
- `Ctrl + Shift + →/←`: Select satu kata sekaligus.

## 7 | Navigasi Cursor dengan Cepat

Walaupun menggunakan cursour memang sudah cepat secara alaminya, namun ketika kita mengetik, lalu ingin berpindah kursor tidak terlalu jauh, waktu yang dihabiskan untuk menggerakan tangan ke mouse lalu kembali lagi ke keyboard kadang memakan waktu yang lebih banyak.

Ada beberapa trick shortcut keyboard untuk navigasi cursour:

- `Ctrl + →/←`: Melompat antar kata
- `Ctrl + ↑/↓`: Melompat antar paragraf
- `Home`: Pindah ke baris awal
- `End`: Pindah ke akhir baris
- `Ctrl + Home`: Pindah ke awal dokumen (paling atas)
- `Ctrl + End`: Pindah ke akhir dokumen (paling bawah)

## 8 | Cut / Memindahkan

Untuk memindahkan teks atau file, kita bisa menggunakan shortcut cepat dengan menggunaka tombol `Ctrl + X`, lalu untuk menaruhnya, kita bisa menggunakan cara yang sama untuk paste biasa, yaitu `Ctrl + V`.

## 9 | Paste tanpa Format

Biasanya ketika kita mempaste teks dari sumber teks tertentu, terdapat format yang ikut terbawa. Untuk menghilangkan format ini, dan memastikan teks hasil paste ini bebas format, maka bisa menggunakan shortcut `Ctrl + Shift + V`.
        
## 10 | Shortcut Lainya Windows

Kita bisa menggukana beberapa kombinasi tombol yang bisa digunakan dengan key `Windows`:
- `Windows + X`: Membuka menu administrasi cepat, seperti *Task Manager*, *Device Manager*, *Disk Management*, *Power Options*, dll.

<br/>

- `Windows + S`: Languns membuka *Seach Bar*, dan kita bisa mengetikan nama aplikasi yang ingin kita buka dengan cara mengetikanya disini.

<br/>

- `Windows + N`: Membuka *Notification Center*, tempat dimana notifikasi dan tanggal muncul. Disini kita juga bisa menyalakan fitur *Do Not Disturb*, sehingga tidak ada nmotifikasi yang mengganggu.

<br/>

- `Windows + .`: Membuka menu Emoji, dan Clipboard gabungan.

<br/>

- `Windwos + E`: Membuka jendela *File Explorer*.

<br/>

- `Windows + Shift + S`: Membuka Snipping Tool versi cepat, lalu membolehkan kita melakukan Screenshoot atau rekam layar (*kualitas rekam layar tidak terlalu bagus dengan opsi ini*).

<br/>

- `Windwos + Ctrl + D`: Membuat Virtual Desktoop baru untuk memisahkan konteks kerja, misal coding di satu dekstop, riset di lainya dan sebagainya.

<br/>

- `Windows + Shift + Esc`: Membuka *Task Manager*

<br/>

- `Windows + L`: Mengunci komputer tanpa menutup aplikasi. Berguna ketika meninggalkan meja kerja sementara.

<br/>

- `Windows + M`: Menyembunyikan atau *minimize* semua jendela yang tampil di desktop. Untuk mengembalikanya lagi, tekan `Windwos + Shift + M`. Namun juga digunakan `Windows + D`.

<br/>

- `Windows + R`: Membuka kotak perintah cepat, dimana bisa langsung digunakan untuk perintah seperti *cmd*, *notepad*, *control*, *appwiz.cpl*, *msconfig*, dll.

<br/>

- `Windows + A`: Membuka *Quick Setting Panel* atau panel notifikasi yang biasanya ada dikanan bawah.

## 11 | Shotcut Yang berhubungan dengan Browser

Untuk membuka tab baru, gunakan shortcut `Ctrl + T`, dan menutup tab baru, bisa digunakan `Ctrl + W`.

Untuk membuka tab yang terakhir diclose, bisa gunakan shortcut `Ctrl + Shift + T`. Ini berguna untuk mengatasi ketika kita tidak sengaja menutup jendela tab pada browser.

Untuk berpindah tab ke kanan dan kiri, bisa gunakan `Ctrl + Tab`, dan `Ctrl + Shift + Tab`.
