# Sumber Daya
Berikut beberapa kategori sumber daya (resources) yang dapat membantu kamu belajar dan berlatih Competitive Programming (CP). Setiap kategori berisi rekomendasi situs, buku, tutorial, dan komunitas yang bisa kamu manfaatkan.

---

## 1. Platform/Laman Latihan & Kontes

1. **Codeforces** ([https://codeforces.com](https://codeforces.com/))
    
    - Kontes rutin (Div. 1, 2, 3, Educational)
        
    - Ribuan soal latihan terklasifikasi berdasarkan tingkat kesulitan
        
    - Fitur “Virtual Contest” untuk mengulang kontes lama
        
2. **AtCoder** ([https://atcoder.jp](https://atcoder.jp/))
    
    - ABC (Beginner Contest), ARC (Regular), AGC (Grand)
        
    - Editorial sangat lengkap, contoh solusi, dan checker resmi
        
    - Mode Virtual (ikuti kontes lama pada waktu bebas)
        
3. **CodeChef** ([https://www.codechef.com](https://www.codechef.com/))
    
    - Long Challenge (10 hari), Cook-Off (2 jam), Lunchtime (3 jam)
        
    - Ribuan soal latihan dengan label “star” (1★–6★)
        
    - Forum diskusi berdedikasi dan tutorial khusus
        
4. **HackerRank** ([https://www.hackerrank.com](https://www.hackerrank.com/))
    
    - Kategori latihan: Algorithms, Data Structures, Mathematics, dsb.
        
    - “30 Days of Code”, “Week of Code”, dan berbagai challenge harian
        
    - Sertifikat dan badge yang bisa dipamerkan di profil
        
5. **LeetCode** ([https://leetcode.com](https://leetcode.com/))
    
    - Soal berbasis interview (DSA + CP)
        
    - Weekly/Biweekly Contests (4 soal dalam 90 menit)
        
    - “Discuss” aktif: banyak solusi dan pembahasan lengkap
        
6. **HackerEarth** ([https://www.hackerearth.com](https://www.hackerearth.com/))
    
    - Daily/Weekly Challenges, Hiring Contests (rekrutmen perusahaan)
        
    - Tutorial lengkap per topik (DSA, algoritma, matematika)
        
    - Kurang lebih 1.000+ soal latihan dengan tag jelas
        
7. **SPOJ (Sphere Online Judge)** ([https://www.spoj.com](https://www.spoj.com/))
    
    - Koleksi soal klasik (20.000+), termasuk kategori “Classical” dan “Challenge”
        
    - Banyak problem tuan rumah lama (LTIME, CODECHEF, dsb.)
        
    - Komunitas forum untuk berdiskusi soal sulit
        
8. **TopCoder** ([https://www.topcoder.com](https://www.topcoder.com/))
    
    - SRM (Single Round Match): 3 soal (mudah, sedang, sulit) dalam 1 jam
        
    - Marathon Matches (kontes optimasi dalam beberapa hari)
        
    - Arena pemrograman berbasis web, rating tradisional
        
9. **Kattis** ([https://open.kattis.com](https://open.kattis.com/))
    
    - Banyak soal CP dari ACM ICPC regionals, NWERC, dsb.
        
    - Soal-soal real-time dengan batasan ketat (C++/Java/Python)
        
    - Ranking global per universitas/negara
        
10. **Timus Online Judge** ([https://acm.timus.ru](https://acm.timus.ru/))
    
    - Soal-soal CP klasik (lebih banyak topik matematika)
        
    - Banyak soal dari ACM ICPC dan Rusia National Olympiad
        
11. **UVa Online Judge** ([https://onlinejudge.org](https://onlinejudge.org/))
    
    - Ribuan soal CP klasik (ACM ICPC archive)
        
    - Banyak soal berbasis matematika dan graf
        
    - Butuh daftar uva.onlinejudge.org/index.php?option=onlinejudge&page=show_problem&problem=10 di UVa LiveArchive

## 2. Tutorial & Artikel Teori / Penjelasan Konsep

1. **CP-Algorithms (e-maxx.ru / cp-algorithms.com)**
    
    - Artikel terstruktur:
        
        - Data Structures (segment tree, fenwick, dsb.)
            
        - Graph (BFS/DFS, Dijkstra, MST, flow)
            
        - Math (number theory, combinatorics, geometry)
            
        - String (KMP, Z-algo, suffix array)
            
        - DP (knapsack, bitmask, tree DP)
            
    - Kode C++ siap pakai; penjelasan step-by-step
        
2. **GeeksforGeeks – Mathematical Algorithms** ([https://www.geeksforgeeks.org/mathematical-algorithms/](https://www.geeksforgeeks.org/mathematical-algorithms/))
    
    - Topik Number Theory (GCD, sieve, modular arithmetic)
        
    - Combinatorics (Pascal’s triangle, binomial coefficients)
        
    - Tutorial singkat plus contoh kode (C++/Java)
        
3. **USACO.guide** ([https://usaco.guide](https://usaco.guide/))
    
    - Modul per-topik:
        
        - Number Theory, Combinatorics, Geometry, Data Structures, DSU, DSQ, dsb.
            
    - Contoh kode C++ referensi, soal latihan dari USACO dan Codeforces
        
4. **Codeforces EDU Courses** ([https://codeforces.com/edu/courses](https://codeforces.com/edu/courses))
    
    - Kursus online (teks + video) untuk:
        
        - Data Structures (stack, queue, segment tree)
            
        - Number Theory (modular arithmetic, CRT)
            
        - Combinatorics (precompute factorial, Lucas Theorem)
            
        - DP dasar hingga advanced
            
5. **TopCoder Tutorial – Math & DS/Algo** ([https://www.topcoder.com/community/competitive-programming/tutorials/](https://www.topcoder.com/community/competitive-programming/tutorials/))
    
    - Ringkasan cepat Number Theory, Combinatorics, Geometry
        
    - Pseudocode/C++ snippet praktis
        
6. **AtCoder Tutorial (GitHub mirror & blog)**
    
    - Banyak komunitas menerjemahkan editorial AtCoder (ABC, ARC, AGC)
        
    - Pembahasan problem by problem dalam bahasa Inggris/Indonesia
        
7. **Khan Academy – Computer Science (algorithms)** ([https://www.khanacademy.org/computing/computer-science/algorithms](https://www.khanacademy.org/computing/computer-science/algorithms))
    
    - Video pendek menjelaskan konsep dasar algoritma dan struktur data
        
    - Visualisasi interaktif untuk sorting, searching, recursion
        
8. **VisuAlgo** ([https://visualgo.net](https://visualgo.net/))
    
    - Visual interaktif struktur data & algoritma (tree, graph, sort)
        
    - Animasi jelas; membantu memahami logika internal
        
9. **Brilliant.org** ([https://brilliant.org](https://brilliant.org/))
    
    - Kursus interaktif berbasis bahasa Inggris:
        
        - Number Theory, Combinatorics, Probability, Graph Theory
            
    - Cocok untuk konsep dasar + soal pendek

## 3. Buku (Cetak & E-Book)

1. **“Competitive Programming” Vol. 1 & 2 – Steven Halim & Felix Halim**
    
    - Bab Math: Number Theory, Combinatorics, Geometry
        
    - Bab DS & Algo lengkap, banyak latihan soal
        
2. **“Introduction to Algorithms” (CLRS) – Cormen, Leiserson, Rivest, Stein**
    
    - Standar teori struktur data & algoritma (tunggu edisi terbaru)
        
    - Meski berat di teori, fondasi penting untuk memahami kompleksitas dan proof
        
3. **“Art and Craft of Problem Solving” – Paul Zeitz**
    
    - Fokus pada strategi problem solving:
        
        - Induksi, Pigeonhole, Combinatorial Arguments, Generating Functions
            
    - Banyak contoh soal matematis level Olympiad
        
4. **“Concrete Mathematics” – Graham, Knuth, Patashnik**
    
    - Materi lanjutan: Summation, Recurrences, Binomial Identities, Discrete Probability
        
    - Ideal untuk yang ingin memahami teori di balik CP
        
5. **“Competitive Programming in C++” – Pranjal Bhuyan, et al.**
    
    - Buku ringkas yang berfokus pada implementasi cepat:
        
        - DSU, Graph, Segment Tree, DP, Math
            
6. **“Programming Challenges” – Skiena & Revilla**
    
    - Kumpulan soal CP tingkat “ACM ICPC training”
        
    - Setiap soal diikuti diskusi pendek dan solusi C++
        
7. **“Guide to Competitive Programming” – Antti Laaksonen**
    
    - Ringkasan semua topik penting dengan contoh kode C++
        
    - Cocok untuk review cepat sebelum kontes
        
8. **“Introduction to Number Theory” – Mathew Crawford (AoPS)**
    
    - Seri AoPS, tetapi khusus Number Theory (modular, CRT, teorema Euler, FLT)
        
    - Sangat berguna saat soal CP memerlukan proof-based thinking

## 5. Blog & Artikel Teknis

1. **Codeforces Blog ([https://codeforces.com/blog](https://codeforces.com/blog))**
    
    - Artikel-artikel komunitas CF tentang hack, trik, dan pembahasan masalah tertentu.
        
    - Banyak kontributor membagikan insight optimasi, tutorial, dan review alat bantu (VS Code, template C++, dsb.).
        
2. **GeeksforGeeks ([https://www.geeksforgeeks.org](https://www.geeksforgeeks.org/))**
    
    - Ribuan artikel tentang DS/Algo, CP, dan matematika.

3. **TopCoder Blog ([https://www.topcoder.com/community/competitive-programming](https://www.topcoder.com/community/competitive-programming))**
    
    - Artikel resmi TopCoder, termasuk tutorial DSU, MST, dan berbagai trik time/space optimization.
        
4. **AtCoder Community Mirrors (GitHub & Blog)**
    
    - Banyak kontributor menerjemahkan editorial AtCoder (ABC, ARC, AGC) ke bahasa Inggris/Indonesia.
        
    - Koleksi link blog yang mengulas solusi per-soal, berguna jika kesulitan memahami editorial resmi.
        
5. **USACO.guide Blog ([https://usaco.guide/blog](https://usaco.guide/blog))**
    
    - Ringkasan problem USACO dan diskusi solusi, plus tips persiapan Olimpiade Pemrograman Pemula hingga Gold Division.

## 6. Forum & Komunitas Diskusi

1. **Reddit r/competitiveprogramming ([https://www.reddit.com/r/competitiveprogramming/](https://www.reddit.com/r/competitiveprogramming/))**
    
    - Diskusi soal CP, tips lomba, rekomendasi buku, dan request review kode.
        
    - Cukup aktif; banyak anggota berbagi pengalaman dan trik kontes.
        
2. **Reddit r/csquestions ([https://www.reddit.com/r/csquestions/](https://www.reddit.com/r/csquestions/))**
    
    - Lebih umum ke pertanyaan seputar ilmu komputer, tapi sering ada thread tentang CP dan wawancara teknik.
        
3. **StackOverflow & StackExchange Competitive Programming**
    
    - Di StackOverflow sendiri banyak tanya‐jawab tentang bug kode CP.
        
    - Di StackExchange ada subforum Code Golf (tidak murni CP), tapi kadang ada diskusi trik optimasi.
        
4. **Discord & Telegram Grup Lokal/Global**
    
    - Banyak grup Discord/Telegram di setiap negara (misalnya “Indonesia Competitive Programming”), tempat berdiskusi soal, berbagi jadwal kontes, dan challenge harian.
        
    - Cari link undangan di postingan Codeforces/Reddit lokal CP.
        
5. **ICPC Regional Community**
    
    - Banyak regional (Asia Pacific, Eropa, Amerika) punya grup/kanaI Discord untuk tim ICPC—cocok jika fokus ke persiapan tim.

## 7. Alat & Plugin Pendukung Coding

1. **VS Code dengan CP Extensions**
    
    - Extensions:
        
        - _Competitive Programming Helper_ (generate template, compile & run cepat)
            
        - _Code Runner_ (mengeksekusi potongan kode langsung di editor)
            
    - Snippet C++:
        
        - _CP Snippets_
            
        - _C++ Intellisense_
            
2. **CLion (by JetBrains)**
    
    - Banyak fitur debugging dan refactoring, cocok jika kamu ingin IDE lebih lengkap dibanding editor ringan.
        
    - Plugin “Competitive Programming” untuk generate template, mengelola test case, dsb.
        
3. **Online IDE untuk Latihan Cepat**
    
    - **repl.it**, **Ideone**, **CodeSandbox** (banyak bahasa didukung, tapi jaringan kadang lambat).
        
    - **Codeforces’ custom invocation**: setting custom invocation agar test case input/output bisa langsung dicek.
        
4. **Kattis Local Runner** ([https://github.com/Kattis/local-runner](https://github.com/Kattis/local-runner))
    
    - Jalankan soal Kattis secara lokal dengan memuat test file resmi—membantu latihan offline.

## 8. Repositori & Koleksi Soal Terbaik

1. **Codeforces Problemset Archive**
    
    - Gunakan filter:
        
        - Difficulty (800–3500)
            
        - Tags (greedy, graph, dp, dsu, math, ds)
            
    - “Rated by tag” feature untuk mengukur kekuatan di tiap topik.
        
2. **AtCoder Library Practice Contest (ACL Practice)** ([https://atcoder.jp/contests/practice2](https://atcoder.jp/contests/practice2))
    
    - Kumpulan soal yang menguji penggunaan atcoder library (fenwick, segtree, dijkstra, dsu).
        
3. **CSES Problemset** ([https://cses.fi/problemset/](https://cses.fi/problemset/))
    
    - 300+ soal CP dasar dan lanjutan:
        
        - Sorting & Searching, Graph Algorithms, Tree Algorithms, DP, Geometri, dsb.
            
    - Setiap soal punya kumpulan solusi dari pengguna.
        
4. **USACO Training Gateway** ([http://www.usaco.org/index.php?page=viewproblem2&cpid=771](http://www.usaco.org/index.php?page=viewproblem2&cpid=771))
    
    - Soal-soal USACO terbagi per tingkatan (Bronze, Silver, Gold), bagus untuk kenali pola soal kontes Amerika.
        
5. **Project Euler** ([https://projecteuler.net](https://projecteuler.net/))
    
    - 800+ soal matematika-programming, fokus pada trik matematika & optimasi.
        
    - Banyak CP’er menggunakan ini untuk melatih teori bilangan dan kombinatorial.
        
6. **LeetCode Explore & Interview Prep**
    
    - Selain contest, LeetCode punya “Explore” modul (algorithms, databases, concurrency) dan set soal Interview; beberapa crossover dengan CP.

## 9. Kursus & Bootcamp Online

1. **Coursera: Algorithms Specialization (Stanford)**
    
    - Seri kursus oleh Tim Roughgarden / Stanford University. Mengajarkan struktur data, greedy, divide and conquer, dynamic programming.
        
2. **EDX: Data Structures and Algorithms (UC San Diego, NYU)**
    
    - Berisi video kuliah, kuis, dan proyek, cocok untuk belajar terstruktur.
        
3. **Udemy: Master the Coding Interview: Data Structures + Algorithms**
    
    - Meski fokus interview, materi DS/Algo banyak yang relevan dengan CP.
        
4. **PluralSight: C++ Performance Critical Programming**
    
    - Pengoptimalan kode C++ (memori, kecepatan)—berguna untuk CP yang butuh micro-optimizations.
        
5. **USACO Online Training**
    
    - Konten kursus gratis resmi USACO, mencakup DS/Algo standar dan soal latihan.

###  Cara Memanfaatkan Semua Sumber Daya di Atas

1. **Jadwal Harian/Mingguan:**
    
    - Tentukan waktu studi:
        
        - 30–60 menit membaca teori (CP-Algorithms, GfG, AtCoder blog)
            
        - 60–120 menit latihan soal (Codeforces/AtCoder)
            
        - 30 menit review editorial & diskusi
            
2. **Target Spesifik:**
    
    - Pilih topik (DP, graph, math) setiap minggu, gunakan sumber tutorial + soal tag terkait.
        
    - Misalnya, minggu 1 fokus “graph” → baca modul USACO.guide + CP-algorithms → latih soal graph di CSES/CF.
        
3. **Catat dan Evaluasi:**
    
    - Simpan kode template di satu file, buat catatan singkat tentang trik atau algoritma baru.
        
    - Gunakan GitHub/Drive untuk backup catatan, supaya mudah dicari kembali.
        
4. **Kolaborasi & Diskusi:**
    
    - Gabung komunitas lokal/global, sampaikan kesulitan, diskusikan solusi alternatif.
        
    - Ikut virtual contest bersama teman, lalu diskusikan setiap soal sesudah kontes.
