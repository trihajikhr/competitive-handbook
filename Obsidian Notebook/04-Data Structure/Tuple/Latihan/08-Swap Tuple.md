# Swap Tuple

Tugas:

1. Buat dua tuple `a` dan `b`, misalnya:
    
    ```cpp
    tuple<int, char> a{1, 'X'}, b{2, 'Y'};
    ```
    
2. Tukar isi `a` dan `b` menggunakan `std::swap`.
    
3. Cetak isi keduanya setelah ditukar.
