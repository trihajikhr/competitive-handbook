# 1 | Latihan Dasar Linked List 

## 1.1 | Latihan Awal

Buatlah sebuah Linked list, dimana kamu harus menambahkan 3 data nama sebagai berikut:
- Faridh Dun Dun
- Toni Alias
- Jacob Marco

Setelah membuat 3 nama tersebut, lakukan beberapa operasi penamabahan dan penyisipan sebagai berikut:

1. Lakukan penambahan dibelakang, yaitu dengan nama `alex`
2. Lakukan penamabahn di depan, dengan nama `lisna`.
3. Lakukan penyisipan, tepat setelah Jacob marco, yaitu nama `Bellingham`


## 1.2 | Class Node Dasar

Buatlah linked list, dengan menggunakan class, dimana menerapkan consttructor, dan memiliki minimal 3 fungsi yang sudah ada didalamnya. Dimana, 2 fungsi tersebut bersifat wajib, yaitu fungsi tambah belakang `push_back()`, dan juga fungsi `travers()`. Sedangkan satu sisa fungsi, bebas. 

Lalu lakukan inputan secara dinamis, dengan cara memasukan data sebagi berikut:

- 1,2,3,4,5, lalu travers.
- Lakukan penambahan dibelakang yaitu 100, lalu travers
- Lalu jalankan fungsi ketiga.
