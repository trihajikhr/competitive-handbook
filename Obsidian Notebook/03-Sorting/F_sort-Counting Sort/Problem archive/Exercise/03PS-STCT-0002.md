# counting sort standar
Time limit: -
Memory limit: -

DIberikan sebuah array non negatif tak terurut. Urutkan array tersebut dengan urutan ascending menggunakan algoritma counting sort.

### Input
- `n` untuk banyaknya array.
- `arr` untuk menampung nilai elemen;
#### Input constraints
\-
### Output
Array yang sudah terurut
### Examples
Input:
```
5
8 6 0 4 2
```

Output:
```bash
0 2 4 6 8
```

### Note
Counting sort bekerja sangat cepat untuk rentang data kecil dan angka maksimal yang tidak terlalu besar. Cocok untuk array dengan rentang dan batas value yang kecil.
# Jawaban
Berikut jawabanya: 

```cpp
#include<iostream>
#include<vector>
#include<algorithm>
using namespace std;

void countingSort(vector<int>&arr){
    if(arr.empty()) return;

    int maxVal=*max_element(arr.begin(), arr.end());

    vector<int> count (maxVal+1,0);

    for(int num:arr){
        count[num]++;
    }

    int index=0;
    for(int i=0; i<= maxVal; i++){
        while(count[i]>0){
            arr[index++]=i;
            count[i]--;
        }
    }
}

inline void fastio(){
   ios_base::sync_with_stdio(0);
   cin.tie(0);
   cout.tie(0);
}

int main() {
    fastio();
    
    int n; cin >> n;
    vector<int>arr(n);
    for(int&x:arr) cin >> x;

    countingSort(arr);

    for(int&x:arr) cout << x << " ";
    cout << "\n";

    
    return 0;
}
```

### Fungsi Counting Sort:

```cpp
void countingSort(vector<int>& arr)
```

- Fungsi `countingSort` menerima **referensi ke vector** `arr` agar perubahan langsung berdampak pada array asli (tanpa salinan).
    

---

#### Pengecekan awal:

```cpp
if (arr.empty()) return;
```

- Cek jika array kosong, langsung keluar dari fungsi (tidak perlu diproses).
    

---

#### Mencari nilai maksimum:

```cpp
int maxVal = *max_element(arr.begin(), arr.end());
```

- `max_element` mencari nilai maksimum dalam `arr`.
    
- `*` digunakan karena `max_element` mengembalikan **iterator**, bukan langsung nilainya.
    

---

#### Membuat dan menginisialisasi array count:

```cpp
vector<int> count(maxVal + 1, 0);
```

- `count` adalah array bantu untuk menyimpan jumlah kemunculan setiap angka.
    
- Ukurannya `maxVal + 1` karena kita menghitung dari `0` sampai `maxVal`.
    
- Diinisialisasi semua nilai menjadi `0`.
    

---

#### Mengisi array count:

```cpp
for (int num : arr) {
    count[num]++;
}
```

- **Loop ini menghitung frekuensi** setiap elemen dalam `arr`.
    
- Misal `arr` mengandung `2` tiga kali, maka `count[2]` akan bernilai 3.
    

---

#### Menyusun kembali array:

```cpp
int index = 0;
for (int i = 0; i <= maxVal; ++i) {
    while (count[i] > 0) {
        arr[index++] = i;
        count[i]--;
    }
}
```

- Loop `for` ini membaca `count` dari 0 sampai `maxVal`.
    
- Selama `count[i] > 0`, kita masukkan `i` ke array `arr` dan kurangi `count[i]`.
    
- Ini menyusun `arr` secara urut.
