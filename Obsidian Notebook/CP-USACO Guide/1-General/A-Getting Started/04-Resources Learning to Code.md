# Resources Learning to Code
## General

- [Sololearn](https://www.sololearn.com/)
	Kursus gratis tersedia untuk C++, Java, dan Python 3. Namun, beberapa fitur memerlukan peningkatan ke versi Pro.

<br/>

- [Codeacademy](https://www.codecademy.com/)
	Kursus gratis tersedia untuk C++, Java, dan Python 3. Namun, beberapa fitur memerlukan peningkatan ke versi Pro.

<br/>

- [w3schools](https://www.w3schools.com/)
	Kursus gratis tersedia untuk C++, Java, dan Python 3

## C++
Gunakan salah satu sumber di atas atau di bawah (atau cari sumbermu sendiri) untuk belajar C++. Jika kamu menggunakan Sololearn, kamu tidak perlu menyelesaikan seluruh kursus; kami merekomendasikan agar kamu menyelesaikan semua materi hingga (dan termasuk) bagian "Functions".


> [!NOTE] Resources
> - [C++ Language Tutorial](http://www.cplusplus.com/doc/tutorial/) -> Untuk pengenalan.
> - [Learn C++](https://www.learncpp.com/) -> Lebih mendalam dibandingkan sumber di atas. Berisi panduan lengkap untuk pengaturan Visual Studio Community (Windows) dan Code::Blocks (Windows, Linux).
> - [2 - Programming in C++](https://usaco.guide/PAPS.pdf#page=29) -> Panduan pengaturan Visual Studio Code untuk Windows, Linux, dan Mac, disertai contoh kode dan latihan soal dari Kattis.
