# Expected Knowledge
## Expected Knowledge

Panduan ini mengasumsikan bahwa kamu sudah memahami topik-topik berikut dalam salah satu bahasa pemrograman C++, Java, atau Python:

- Variabel
    
- Tipe data
    
- Membaca input
    
- Menulis output
    
- Perulangan (Loops)
    
- If / Else
    
- Operator logika
    
- Fungsi
    
- Rekursi dasar (fungsi yang memanggil dirinya sendiri)
    
- Array
    
- Array multidimensi

> [!NOTE]  Tips Pro  
Untuk C++, pengetahuan tentang pointer, struct, dan class memang berguna, tetapi tidak wajib.

## Getting Started
Sebaiknya kamu mulai dengan soal-soal yang lebih pendek daripada soal-soal yang biasanya muncul di **USACO Bronze**. Sumber-sumber berikut mungkin bisa membantu kamu untuk lebih mengenal bahasa pemrograman pilihanmu:

- CodeSignal: [Arcade, Interview Practice](https://app.codesignal.com/) -> no need to handle input or output
- VPlanet: [Learn to Code](https://www.vplanetcoding.com/course1-usaco-basics) -> basic problemsets with and without nested loops
- CSES: [Introductory Problems](https://cses.fi/problemset/list/) -> up to and including "Palindrome Reorder"
