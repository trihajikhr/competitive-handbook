# Running Code Online
## Running Code Online

Tersedia untuk semua bahasa: C++, Python, dan Java. Semua opsi ini gratis.
### [ide.usaco.guide](https://ide.usaco.guide/)

- Mendukung kolaborasi waktu nyata (cukup bagikan URL yang lengkap dengan bagian belakangnya).
    
- Jumlah percobaan menjalankan kode terbatas, jadi jangan terlalu sering menekan tombol "Run" secara berlebihan. Laporkan masalah di [Github](https://github.com/cpinitiative/ide)
    
- Kadang tidak stabil; jika mulai bermasalah, buat file baru saja.
    
- Ini adalah versi terbaru dari situs sebelumnya.

### [ide.judge0.com](https://ide.judge0.com/)

- Pada dasarnya mirip dengan yang di atas, tapi tanpa fitur kolaborasi waktu nyata dan lebih stabil.

### [Ideone](http://ideone.com/)

- Menyediakan fitur dasar yang dibutuhkan untuk menjalankan kode.
    
- Kadang menghapus kode saat pertama kali dibuat, jadi biasakan salin kode kamu terlebih dahulu sebelum mengetik.
    

### [CSAcademy Workspace](https://csacademy.com/workspace/)

- Cukup bagus (kecuali kalau muncul pesan seperti “Estimated Queue Time: …”).
    
- Jika tertulis “Saved locally”, kode kamu tidak akan tersimpan jika tab ditutup — tekan Command+S (atau Ctrl+S) untuk menyimpan.
    

### [repl.it](https://repl.it/)

- Mendukung kolaborasi waktu nyata.
    
- Perlu berlangganan berbayar untuk membuat repl pribadi.
    

### [goormIDE](https://ide.goorm.io/)

- Juga mendukung kolaborasi waktu nyata.
    

Kamu juga bisa membagikan kode lewat [pastebin](https://pastebin.com/) atau [hastebin](https://hastebin.com/) jika hanya ingin berbagi cepat.

---

> [!NOTE] ⚠️ Peringatan!  
Kode kamu tidak boleh bisa dilihat publik selama kontes berlangsung. Sebagai contoh, jika kamu menggunakan Ideone, pastikan untuk mengatur kode kamu menjadi secret atau private.
