# Templater Problem Statement
## ID
Dalam pembuatan Id, harus menyertakan menyertakan aturan berikut:

Misal: 03PS-STSWE0001

- **03PS-ST** : Menjabarkan bahwa **P**roblem **S**tatement berkaitan dengan materi bab 3, yaitu **S**or**T**ing
- **SW** : Menjabarkan bahwa problem berkaitan dengan sub materi tertentu, misal adalah **S**liding **W**indow
- **E** : Problem statement merupakan jenis **E**xcercise, atau latihan biasa
- **0001** : Menjabarkan latihan ke sekian.
# Templater Algoritma
## ID
Untuk Id di materi algoritma, berikut penjelasanya:

Misal: 03TH-STSW001

- **03TH-ST** : Menjabarkan bahwa catatan merupakan pembahasan **TH**eory materi bab 3, yaitu **S**or**T**ing
- **SW** : Menjabarkan bahwaa materi yang sedang dibahas adalah sub materi tertentu, yaitu misal **S**liding **W**indow.
- **001** : menjabarkan materi atau catatn ke sekia.
