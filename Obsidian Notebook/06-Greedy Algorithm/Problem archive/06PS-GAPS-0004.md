# Dua Rumah Terjauh dengan Warna Berbeda
Time limit: -
Memory limit: -

Terdapat **n** rumah yang berjajar rapi di sepanjang jalan, dan setiap rumah dicat dengan indah. Kamu diberikan array integer **colors** dengan panjang **n** (menggunakan indeks 0), di mana `colors[i]` merepresentasikan warna dari rumah ke-i.

**Kembalikan jarak maksimum antara dua rumah yang memiliki warna berbeda.**

Jarak antara rumah ke-i dan ke-j dihitung dengan rumus `abs(i - j)`, di mana `abs(x)` adalah nilai absolut dari x.

---

### Contoh:
![[06PS-GAPS-0004-1.png]]

#### Contoh 1:

```
Input: colors = [1,1,1,6,1,1,1]
Output: 3
```

**Penjelasan:**

Warna 1 adalah biru, dan warna 6 adalah merah.
Dua rumah terjauh dengan warna berbeda adalah rumah ke-0 dan rumah ke-3.
Rumah ke-0 berwarna 1, rumah ke-3 berwarna 6.
Jarak antara keduanya adalah `abs(0 - 3) = 3`.
Catatan: rumah ke-3 dan ke-6 juga bisa menghasilkan jawaban optimal.


#### Contoh 2:
![[06PS-GAPS-0004-2.png]]

```
Input: colors = [1,8,3,8,3]
Output: 4
```

**Penjelasan:**

Warna 1 adalah biru, warna 8 adalah kuning, warna 3 adalah hijau.
Dua rumah terjauh dengan warna berbeda adalah rumah ke-0 dan rumah ke-4.
Jaraknya adalah `abs(0 - 4) = 4`.


#### Contoh 3:

```
Input: colors = [0,1]
Output: 1
Penjelasan:
Dua rumah memiliki warna berbeda dan hanya berjarak 1.
```

---

### Batasan:

- `n == colors.length`
    
- `2 <= n <= 100`
    
- `0 <= colors[i] <= 100`
    
- Data uji dijamin mengandung setidaknya dua rumah dengan warna yang berbeda.

# Jawaban
Jawaban yang sudah cukup efektif:

```cpp
class Solution {
public:
    int maxDistance(vector<int>& colors) {
        int n=colors.size();
        int left=0, rigth=n-1;

        while(colors[0] == colors[rigth]){
            rigth--;
        }

        while(colors[n-1] == colors[left]){
            left++;
        }

        return max(rigth, n-1-left);
    }
};
```

Time complexity: $O(n)$

# Editorial 

### ğŸ§  Ide Utama (Greedy Thinking)

Kita ingin **mencari dua rumah dengan warna berbeda** yang **jaraknya paling jauh**.

Strateginya:

- Kita bisa mencoba membandingkan **rumah pertama** (`colors[0]`) dengan semua rumah dari kanan (`colors[n-1]` ke arah kiri).
    
- Juga sebaliknya, bandingkan **rumah terakhir** (`colors[n-1]`) dengan rumah dari kiri (`colors[0]` ke arah kanan).
    
- Begitu kita menemukan rumah yang warnanya berbeda, kita hitung jaraknya. Karena kita mulai dari ujung, maka jarak yang pertama kita temukan **pasti maksimum**.
    

---

### âœ… Langkah-langkah

1. Misalkan `n` adalah panjang array `colors`.
    
2. Inisialisasi `maxDistance = 0`.
    
3. Bandingkan:
    
    - Dari kiri (`colors[0]`) dengan semua rumah dari kanan.
        
    - Dari kanan (`colors[n-1]`) dengan semua rumah dari kiri.

### ğŸ’¡ Implementasi C++

```cpp
int maxDistance(vector<int>& colors) {
    int n = colors.size();
    int left = 0, right = n - 1;

    // Coba cari dari kiri ke kanan: bandingkan dengan rumah terakhir
    while (colors[0] == colors[right]) {
        right--;
    }

    // Coba cari dari kanan ke kiri: bandingkan dengan rumah pertama
    while (colors[n - 1] == colors[left]) {
        left++;
    }

    return max(right, n - 1 - left);
}
```
