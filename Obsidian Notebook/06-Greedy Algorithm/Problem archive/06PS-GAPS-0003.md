# Jumlah Minimum Gerakan untuk Mengubah Semua Karakter Menjadi 'O'
Time limit: -
Memory limit: -

Kamu diberikan sebuah string `s` yang terdiri dari `n` karakter yang masing-masing adalah 'X' atau 'O'.

Sebuah **gerakan** didefinisikan sebagai memilih tiga karakter berturut-turut dari string `s` dan mengubahnya menjadi 'O'. Perhatikan bahwa jika sebuah gerakan diterapkan pada karakter 'O', karakter tersebut tetap sama.

Kembalikan **jumlah minimum gerakan yang diperlukan** sehingga semua karakter dalam string `s` diubah menjadi 'O'.

---

### Contoh 1:

**Input:**

```
s = "XXX"
```

**Output:**

```
1
```

**Penjelasan:** XXX -> OOO  
Kita memilih ketiga karakter dan mengubahnya menjadi 'O' dalam satu gerakan.

---

### Contoh 2:

**Input:**

```
s = "XXOX"
```

**Output:**

```
2
```

**Penjelasan:** XXOX -> OOOX -> OOOO  
Kita memilih tiga karakter pertama pada gerakan pertama dan mengubahnya menjadi 'O'.  
Kemudian kita memilih tiga karakter terakhir dan mengubahnya sehingga string akhir berisi semua 'O'.

---

### Contoh 3:

**Input:**

```
s = "OOOO"
```

**Output:**

```
0
```

**Penjelasan:** Tidak ada karakter 'X' dalam string `s` untuk diubah.

---

### Batasan:

- `3 <= s.length <= 1000`
    
- `s[i]` adalah 'X' atau 'O'.
    

# Jawaban
Jawaban yang sudah cukup efisien:

```cpp
class Solution {
public:
    int minMovesToConvert(string s) {
        int moves = 0;
        int n = s.length();
        
        for (int i = 0; i < n; ) {
            if (s[i] == 'X') {
                moves++; 
                i += 3;   
            } else {
                i++;      
            }
        }
        
        return moves;
    }
};
```

Time complexity: $O(n)$
Memory complexity: $O(1)$

# Editorial 

Untuk mengubah semua karakter dalam string `s` menjadi 'O', kita perlu mencari cara untuk mengubah sebanyak mungkin 'X' dengan jumlah gerakan yang minimal. Setiap gerakan mengubah tiga karakter berturut-turut menjadi 'O'. Oleh karena itu, kita ingin memilih segmen-segmen 'X' yang panjangnya 3 atau lebih, dan mengubahnya.

## Langkah-langkah Penyelesaian:

1. **Iterasi String**: Mulai dari kiri ke kanan.
    
2. **Identifikasi Segmen 'X'**: Jika kita menemui 'X', kita cek apakah ada tiga karakter berturut-turut yang dapat diubah menjadi 'O'.
    
3. **Lakukan Gerakan**: Setiap kali kita menemui 3 atau lebih 'X' berturut-turut, lakukan gerakan untuk mengubahnya.
    
4. **Lewati Karakter yang Telah Diubah**: Setelah mengubah karakter, lompat tiga posisi ke depan, karena karakter tersebut sudah diubah menjadi 'O'.
    

## Implementasi Solusi:

```cpp
class Solution {
public:
    int minMovesToConvert(string s) {
        int moves = 0;
        int n = s.length();
        
        // Iterasi seluruh string
        for (int i = 0; i < n; ) {
            // Jika kita menemukan 'X', mulai lakukan perubahan
            if (s[i] == 'X') {
                moves++; // Lakukan satu gerakan
                i += 3;   // Lompati tiga karakter yang baru saja diubah
            } else {
                i++;      // Jika bukan 'X', lanjutkan ke karakter berikutnya
            }
        }
        
        return moves;
    }
};
```

### Penjelasan:

- **Iterasi String**: Kita mulai iterasi dari kiri ke kanan.
    
- **Deteksi 'X'**: Jika menemukan 'X', kita lakukan satu gerakan untuk mengubah tiga karakter berturut-turut menjadi 'O'. Setelah itu, kita lompat tiga posisi, karena kita sudah mengubah ketiganya.
    
- **Jumlah Gerakan**: Setiap kali kita mengubah tiga 'X' berturut-turut, kita menambah jumlah gerakan (`moves`).
    

### Kompleksitas:

- **Waktu (Time Complexity)**: O(n), di mana n adalah panjang string. Kita hanya perlu sekali iterasi melalui string.
    
- **Memori (Space Complexity)**: O(1), karena kita hanya menggunakan beberapa variabel tambahan untuk perhitungan, tanpa struktur data tambahan.
